# Ladzik - remote controlled robot using web browser
<p>robot named "Ladzik" is really just a raspberry pi on crawler platform. It has an attached a raspberry pi camera v2 on front and a lipo battery mounted underneath a platform for vision and power.</p> 

<h2 style="color: red">This branch has the code edited so that it can be run on a raspberry pi without a camera - all mention of the camera has been removed, and the livefeed on the site has a great video of Ladzik's future mission.</h2>
