# Ladzik - remote controlled robot using web browser
<p>robot named "Ladzik" is really just a raspberry pi on crawler platform. It has an attached a raspberry pi camera v2 on front and a lipo battery mounted underneath a platform for vision and power.</p> 


<h2 style="color:red">This branch has the code edited so that it can be run on a raspberry pi without a camera and motor driver HAT - all mention of the camera, HAT has been removed, and the livefeed on the site has a great video of Ladzik's future mission. Sterring buttons do nothing</h2>
<p>The implications of these changes go as far as the performance as well as the size on disk of the entire code. This makes merging this branch to the main branch imposible and will never happen.Once the entire project is finished this branch WILL BE DELETED</p>
